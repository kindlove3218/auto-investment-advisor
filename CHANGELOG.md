## 版本说明

### 版本号格式
采用语义化版本控制 (Semantic Versioning)：`MAJOR.MINOR.PATCH`

- **MAJOR**：不兼容的 API 变更
- **MINOR**：向后兼容的新功能
- **PATCH**：向后兼容的 bug 修复

### 更新类型
- **新增**：新功能
- **改进**：现有功能的增强
- **修复**：bug 修复
- **移除**：删除的功能
- **安全**：安全相关的更新
